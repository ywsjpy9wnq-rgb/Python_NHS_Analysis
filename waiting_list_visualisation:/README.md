# Visualising the Relationship Between Total Waiting List and Patients Waiting ≤18 Weeks

## Context
The NHS 18‑week target is a key performance indicator. We examine whether trusts with larger waiting lists also have more patients waiting within the target.

## Data
`nhs_waiting_lists.csv` contains columns `total_waiting_list` and `up_to_18_weeks`.

## Approach
Create a scatter plot with `total_waiting_list` on the x‑axis and `up_to_18_weeks` on the y‑axis. Add a trend line and interpret.

## Findings
A strong positive correlation exists – larger waiting lists are associated with more patients achieving the 18‑week standard. No extreme outliers.
