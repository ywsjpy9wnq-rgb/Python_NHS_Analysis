# NHS Data Analysis - Python

Six Python projects using real NHS waiting lists and staffing data.

## Projects

1. **Regex Waiting Times** – Extract waiting times from text messages
2. **Top NHS Waiting Lists** – Find organisations with largest waiting lists
3. **Waiting List Visualisation** – Scatter plot of waiting lists vs timely care
4. **Ambulance Staff & Doctors** – Compare doctor numbers by ambulance staff
5. **NHS Table Summary** – Merge datasets and create summary table
6. **Logistic Regression** – Predict long waits from staffing levels

## How to Run

1. Clone this repository
2. Install required Python packages:
   ```bash
   pip install pandas matplotlib seaborn statsmodels
   ```
3. Open any `.ipynb` notebook in Jupyter
4. Run all cells
